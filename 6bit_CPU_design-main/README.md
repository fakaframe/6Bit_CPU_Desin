# 6-Bit CPU Design (Computer Architecture)

### Key Specifications:
1. **CPU Word Size**: 6 bits
2. **ALU Operations**: XOR, ADD, SHR
3. **Number of Registers**: 4
4. **RAM Size**: 8 units
5. **Word Size (ISA & RAM)**: 18 bits
6. **Supported Instructions**: Register Mode, Immediate Mode, JMP, JE
